DiabloDrops(更多掉落)
===========

DiabloDrops is a plugin that randomly modifies the equipment of mobs in Minecraft.

In order to use this plugin, you must be running a 1.5.x+ version of CraftBukkit.

Development Team: deathmarine and Nunnery
